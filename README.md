Projeto em Python, criado durante uma aula do curso de pyhton. O intuito maior do projeto é poder colocar todo nosso aprendizaddo do curso, até o momento em um jogo da forca.

O projeto se trata de um simples jogo da forca, onde o objetivo do usuário é conseguir acertar a palavra secreta. O usuário tem um total de 5 (cinco) chances para tentar advinhar, se caso o usário não conseguir, ele perde o jogo.
